# 📚 PHP-Estudo

Repositório criado para estudos da linguagem **PHP**, contendo exercícios, scripts e pequenos projetos desenvolvidos durante o aprendizado.

O objetivo deste projeto é evoluir gradualmente nos conceitos da linguagem, praticando desde o básico até funcionalidades mais avançadas.

---

## 🚀 Tecnologias utilizadas

- PHP (CLI)
- Terminal / Linha de comando
- VS Code

---

## 📌 Conteúdos abordados

- Variáveis e tipos de dados
- Estruturas condicionais (`if`, `elseif`, `else`)
- Estruturas de repetição (`for`, `while`, `do while`)
- Arrays
- Entrada de dados via terminal (`fgets`, `$argv`)
- Operadores lógicos e matemáticos
- Estrutura `match`
- Scripts interativos no terminal (menus)

---

## 💻 Exemplos de funcionalidades

- 🎬 Sistema de avaliação de filmes
- 📊 Cálculo de média de notas
- 🏦 Menu interativo de conta bancária (saque e depósito)
- 🧠 Exercícios de lógica com PHP
